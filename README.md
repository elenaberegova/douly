# Markdown Editor

A lightweight browser-based Markdown editor.

The application provides a writing area, a live preview panel and simple formatting tools. Notes are automatically saved in the browser using Local Storage.

---

## Features

- Live preview
- Toolbar shortcuts
- Auto-save
- Load sample document
- Clean responsive layout
